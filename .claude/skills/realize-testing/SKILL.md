1. **Funcionalidad de la Skill**: Esta skill está diseñada para realizar testing unitario en el backend utilizando JEST. Proporciona comentarios detallados en cada función de test para explicar qué se está testeando y por qué es importante.
2. **Cobertura de Tests**: Asegúrate de cubrir todas las funciones importantes del backend, incluyendo aquellas relacionadas con la gestión de turnos, usuarios y cualquier otra funcionalidad relevante. Esto garantizará que el backend esté bien probado y que cualquier cambio futuro no rompa funcionalidades existentes.
3. **Ejemplos de Ejecución de Tests**: Incluye ejemplos claros de cómo ejecutar los tests utilizando el comando `npm run test`. Proporciona instrucciones sobre cómo interpretar los resultados de los tests, incluyendo qué hacer en caso de que algún test falle y cómo solucionar posibles problemas.
